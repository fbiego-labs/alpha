# alpha

A prototyping board based on Atmel 32u4 microcontroller. 

![alpha](images/alpha_board_v2.jpg?raw=true "board")


- Atmel 32u4 IC
(Arduino Micro bootloader)
- USB Type C

### Features
- 3 LEDs (Red, Yellow, Green)
- 2 Buttons
- 2 Potentiometers
- Buzzer
- LDR
- Infrared transmitter & receiver
- NTC Thermistor

### Breakouts
- 2 Servo
- Serial1
- NRF24L01
